CalendarAccess¶

class calendaraccess.CalendarUtils¶

class CalendarUtils for all calendar methods used to connect with google calendar.

createEvent(user, userEmail, rego, startDate, endDate)¶

takes variables user,userEmail, rego and dates to create a new event on companies google calendar, using credentials stored in MasterPi folder

deleteEvent(event)¶

removes given event from companies google calendar

getCalendarID()¶

gets calendar ID link to output to user to allow them to access that given booking on their calendar
